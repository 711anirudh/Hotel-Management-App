# Hotel Room Booking System

A simple hotel room booking application developed using Java and Java Swing. This project allows users to add rooms, view room details, book rooms, and checkout rooms through a graphical user interface.

## Features
- Add hotel rooms
- View room details
- Book available rooms
- Checkout booked rooms
- Simple GUI using Java Swing

## Technologies Used
- Java
- Java Swing
- Object-Oriented Programming (OOP)

## Project Structure
- `Room.java` – Stores room information and booking status
- `RoomManager.java` – Handles room operations and management
- `Main.java` – Runs the application and GUI

## What I Learned
- Basics of Java GUI development
- Applying OOP concepts in a real project
- Managing application flow and user interaction
- Organizing Java classes and methods

## How to Run

1. Clone the repository
2. Open the project in Eclipse or any Java IDE
3. Run `Main.java`
4. Use the GUI to manage hotel room bookings
